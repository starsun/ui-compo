alert弹出浮层

alert配置参数

参数 默认值 可选参数 说明
width 300   宽度(int)，可传
height 自适应   高度(int)，可传
mask false   是否整个页面有遮罩浮层。true：有，false：无。
content     内容(string)，必传
title     标题(string)，不传整个顶部都没有，可传
type   INFO/WARN/ERROR/SUCC/HELP 类型(string)，可传
 onClose      回调方法(function)，可传
 label  确 定    “确定”的替换文件( string)，可传

<script type="text/javascript">
	$('#alert-demo').bind('click',function() {
		new ALP.Widget.alert({
			type:'WARN',

		});
	});
</script>


confirm弹出浮层

配置参数

参数 默认值 可选值 说明
width 180   宽度(int)，可传
mask true   是否整个页面有遮罩浮层。true：有，false：无。
content     内容(String)，必传
type   INFO/WARN/ERROR/SUCC/HELP 类型(string)，主要区分Icon，可传
onConfirm     当确定时回调 (function)，可传
onCancel     当取消时回调 (function)，可传
 labelConfirm  确定   替换 “确定”，可传
 labelCancel  取消    替换“取消”，可传

这是demo，请点击下面的链接

click me~

这是demo代码

<script type="text/javascript">
	$('#confirm-demo').bind('click',function() {
		new ALP.Widget.confirm({
			title:"确认删除吗?",
            content:"确认删除吗?删除后将不可恢复。",
            type:'WARN',
            onConfirm:function(){
				alert('你点了确定~');
			},
			onCancel:function(){
				alert('你点了取消~');
			}
		});
	});
</script>


自定义弹出浮层

配置参数

参数 默认值 可选值 说明
width     宽度
height     高度
zIndex      
title     标题(必选)
content     内容(必选)
base #bp-box   dialog的父级元素
border     边框
mask     bool true 默认 true:全窗口覆盖，false只有窗体背后一点
padding 4   内边距，默认4 外边框阴影宽度
center     是否居中(bool)
noclose  false  false,true (bool) 没有关闭按钮
